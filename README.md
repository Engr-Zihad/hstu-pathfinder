# HSTU Pathfinder

An AI-powered academic guide for CSE students at Hajee Mohammad Danesh Science and Technology University (HSTU), Dinajpur.

## Features
- AI-powered academic assistance
- CGPA calculator
- Curriculum guide
- Career roadmap

## Tech Stack
React + TypeScript + Vite + Tailwind CSS + Supabase + shadcn/ui

## Getting Started
1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your credentials
3. Run `bun install` then `bun dev`
